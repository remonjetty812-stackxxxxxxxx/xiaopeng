# 小鹏画图

一个基于 Web 的 AI 图像生成工具，通过调用 API 接口生成高质量图片。

## 功能特点

- 🤖 支持多种 AI 模型（GPT Image 2 / Nano Banana Pro）
- 📐 多种画幅比例可选（1:1, 16:9, 9:16, 4:3 等）
- 🖼️ 支持 1K/2K/4K 分辨率
- 📤 支持上传参考图片
- 📜 历史记录管理
- 📱 响应式设计，支持移动端

## 使用方法

1. 打开 `index.html` 文件
2. 输入 API Key
3. 填写创意描述（Prompt）
4. 选择模型、比例和分辨率
5. 点击生成按钮
